### 简介
`visualizerAPI`使用`react`框架构建的一个将后端各类工具进行可视化的开源项目，旨在帮助后端开发者快速
进行开发，系统对接，数据库开发，代码审计，架构设计和学习。

### 使用方法
```shell
yarn install
npm run build
```
如果需要在本地使用，则执行 `yarn start`即可。
